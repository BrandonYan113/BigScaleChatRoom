# BigScaleChatRoom
Big scale chat room
校园新闻服务工作站，实现主要其中的功能：基于深度学习的目标识别，用于校园流浪动物的识别工作，大规模人数的聊天室功能，新闻发布功能和捐赠平台。
主要设计语言为Python，Golang和前端UI的HTMl, Javascript/x等。主要使用开发框架包括Pytorch, Nginx, gRPC, React等。
项目目前实现了聊天室的账号注册、登录和发送群消息功能。深度识别项目还未上传。欢迎开源大佬来指导开发
